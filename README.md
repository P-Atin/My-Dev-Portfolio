# 🌟 Portfolio — Atin Patrice

> Développeur Full-Stack · Étudiant SUPINFO

Un portfolio personnel moderne et interactif, conçu pour présenter mes projets, compétences et parcours en tant que développeur full-stack.

---

## ✨ Aperçu

Ce portfolio a été construit avec une attention particulière portée à l'expérience visuelle : animations fluides, curseur personnalisé, icosaèdre 3D interactif et champ de particules en arrière-plan. L'objectif est de refléter ma passion pour le design et le développement web.

---

## 🚀 Stack Technique

| Catégorie | Technologies |
|-----------|-------------|
| Framework | React 18 + TypeScript |
| Build tool | Vite |
| Styling | Tailwind CSS |
| Animations | Framer Motion |
| 3D | Three.js |
| Composants UI | shadcn/ui |
| Icons | Lucide React |

## 📁 Structure du projet
```
src/
├── components/
│   ├── HeroSection.tsx        # Section d'accueil avec effet typewriter
│   ├── AboutSection.tsx       # Présentation personnelle
│   ├── ExperienceSection.tsx  # Timeline du parcours professionnel
│   ├── ProjectsSection.tsx    # Grille des projets réalisés

```

## 🛠️ Installation & Lancement

### Prérequis

- Node.js (v18 ou supérieur)
- npm

### Étapes
```bash
# 1. Cloner le dépôt
git clone https://gitlab.com/patriceatin/portfolio.git
cd portfolio

# 2. Installer les dépendances
npm install

# 3. Lancer en développement
npm run dev
```

Le projet sera accessible sur **http://localhost:5173**

### Autres commandes
```bash
# Build de production
npm run build

# Prévisualiser le build
npm run preview

# Lancer les tests
npm run test
```
